# 🌕 月齢ウィジェット（Moon Phase Widget）

これは、現在の月齢を日本語で表示するシンプルなウィジェットです。  
Notionなどのページに埋め込んで、暦やセルフケアのリズムを整えるために使えます。

## 🔧 使用技術

- HTML / CSS / JavaScript
- GitHub Pages（ホスティング）

## 📅 表示内容

- 今日の日付に対応した月齢（日数）
- 🌑🌒🌓🌔🌕 などのアイコン表示（※予定）
- 和風デザインへのアレンジ予定
